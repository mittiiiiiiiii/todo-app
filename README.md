# TODOアプリ

## 実装

- Next.jsで実装
  - API経由でDBにデータ読み書き
  - ユーザー登録・ログイン
- Mysqlで実装
  - Next.jsではprismaでdbを管理するのが推奨されいるためprismaで実装

## 環境構築

```bash
# 起動
make install

# テーブルをみる
make studio

# ログをみる
make logs
```
